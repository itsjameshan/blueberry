# ONNX 模型说明

本目录存放蓝莓成熟度检测的导出模型，**多个版本来源不同**，部署/汇报时请按下表区分。

| 文件 | 模型 | 训练数据 | 谁汇报 |
|------|------|----------|--------|
| `best.onnx` | YOLO11s | 同学标注的 2700+ 张照片（**未修正前**的版本） | 男班长小罗发布的「组 A 汇报」用的就是这个结果 |
| `v12best.onnx` | YOLO12s | 原始数据集 | 科研组 A 汇报用的是这个结果 |
| `v11best.onnx` | YOLO11s | （见 git 历史 / 训练记录） | — |

## 关键区别

- **`best.onnx`** = 重新训练版。基于同学手工标注的 2700+ 张照片，标注**修正之前**的版本，用 YOLO11s 训练得到。男班长小罗对外发布的组 A 汇报结果就是这个模型。
- **`v12best.onnx`** = 科研组 A 汇报版。用**原始数据集**、YOLO12s 训练得到，和上面那个不是一回事。

⚠️ 两份结果数据来源、骨干网络都不同，引用结论时务必标明用的是哪个模型，避免混淆。
