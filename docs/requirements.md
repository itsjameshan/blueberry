# 蓝莓成熟度检测系统 — 气象模块需求文档

## 一、项目概述

在现有蓝莓成熟度检测系统基础上，新增**气象预警与农业建议模块**，并在两个子系统前增加**门户首页**作为统一入口。

## 三、功能需求

### 3.1 门户首页

- 登录后默认跳转到门户首页
- 展示两个子系统入口卡片：蓝莓检测系统、气象中心
- 显示当前用户信息和退出按钮

### 3.2 果园管理（仅管理员可操作）

| 功能 | 说明 |
|------|------|
| 创建果园 | 填写果园名称、所在地（城市）、种植日期 |
| 编辑果园 | 修改基本信息 |
| 删除果园 | 删除果园及其关联数据 |
| 设置生长阶段 | 手动选择：萌芽期 / 花期 / 果期 / 休眠期 |

### 3.3 气象数据展示（所有用户只读）

| 数据 | 来源API | 展示内容 |
|------|---------|---------|
| 实时天气 | 天气预报 | 温度、湿度、体感温度、风向风速、降水量、气压 |
| 天气预报 | 天气预报 | 未来3~7天逐日预报 |
| 分钟降水 | 分钟降水 | 未来2小时逐分钟降水（短临预警） |
| 天气预警 | 天气预警 | 当前生效的预警信息 |
| 天气指数 | 天气指数 | 紫外线指数、运动指数等 |
| 空气质量 | 空气质量 | AQI、PM2.5、PM10 |
| 天文信息 | 天文 | 日出日落时间、日照时长 |
| 历史天气 | 时光机 | 指定日期的历史天气 |

### 3.4 智能农业建议

**逻辑：蓝莓生长阶段 × 气象条件 → 农业措施建议**

建议等级：
- **正常**（绿色）：气象条件适宜，常规管理
- **注意**（黄色）：气象条件有偏差，需要关注
- **预警**（红色）：极端气象，需立即采取措施

示例规则：

| 生长阶段 | 气象条件 | 建议措施 |
|---------|---------|---------|
| 花期 | 温度 > 30°C | 高温影响授粉，建议遮阳网覆盖、增加喷灌降温 |
| 花期 | 连续阴雨 > 3天 | 湿度过高影响授粉，建议人工辅助授粉、注意灰霉病防治 |
| 花期 | 低温 < 5°C | 霜冻风险，建议熏烟防霜、覆盖保温 |
| 果期 | 温度 > 35°C | 高温灼果，建议遮阳、早晚灌溉、地面覆盖保湿 |
| 果期 | 暴雨预警 | 及时排水防涝、采摘已成熟果实减少损失 |
| 果期 | 大风预警 | 加固支架、提前采摘 |
| 休眠期 | 低温 < -10°C | 极端低温冻害，建议培土、包裹树干、熏烟 |
| 萌芽期 | 倒春寒预警 | 延迟萌芽措施、准备防霜物资 |

### 3.5 预警推送

- 页面内预警：首页/气象页面顶部展示当前生效的预警信息
- 邮件推送：检测到极端气象预警时，自动发送邮件通知果园管理者

## 四、权限设计

| 角色 | 门户首页 | 蓝莓检测系统 | 气象中心 |
|------|---------|------------|---------|
| 管理员 | 可见 | 完全操作 | 果园管理 + 查看气象 |
| 普通用户 | 可见 | 完全操作 | 仅查看气象（只读） |

## 五、页面结构（新增）

| 页面 | 路径 | 说明 |
|------|------|------|
| 门户首页 | `/portal` | 两个子系统入口卡片 |
| 气象中心首页 | `/weather` | 果园列表 + 实时天气概览 + 预警信息 |
| 果园管理 | `/weather/garden` | 创建/编辑/删除果园（仅管理员） |
| 天气预报详情 | `/weather/forecast` | 7天预报、分钟降水、历史天气 |
| 农业建议 | `/weather/advice` | 基于当前生长阶段+气象的智能建议 |
