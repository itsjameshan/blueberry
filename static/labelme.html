操作步骤

Labelme 标注蓝莓成熟度三类（ripe / semi-ripe / unripe）

①

安装并启动

pip install labelme，命令行运行 labelme 打开界面。

②

打开图片目录

点 Open Dir 选择蓝莓图片文件夹，左侧列表逐张标注。

③

画框打标签

Create Polygons / 矩形框选果实，输入标签 ripe / semi-ripe / unripe。

④

保存标注

Ctrl+S 保存，每张图生成同名 .json 标注文件。

⑤

转 YOLO 格式

将 json 转换为 YOLO txt（本项目脚本：split_dataset.py 等）。

⑥

训练模型

用标注好的数据集训练 YOLO11，导出 ONNX 供平台检测。
